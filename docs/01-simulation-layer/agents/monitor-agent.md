# MonitorAgent

## Overview

The `MonitorAgent` collects simulation data for visualization and analysis. It logs all transactions and system states to enable real-time monitoring and post-simulation analysis.

## Responsibilities

- Collects simulation data for visualization
- Logs all transactions and system states
- Aggregates agent state information
- Provides data for dashboard visualization
- Tracks system-wide metrics
