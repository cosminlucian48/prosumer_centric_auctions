# Web Dashboard (Frontend)

The web dashboard provides real-time visualizations and interactive controls for the energy trading simulation.

## Components

- **[Visualizations](./visualizations/)** - Charts, graphs, and visual displays
- **[Controls](./controls/)** - Interactive simulation controls
- **[Components](./components/)** - Reusable UI components

## Responsibilities

- Display real-time visualizations:
  - Energy production/consumption
  - Trade network graph
  - Market prices over time
  - Agent details and battery levels
- Provide controls for simulation input:
  - Slider for weather intensity
  - Add/remove agents
  - Start/stop simulation
  - Parameter adjustment
