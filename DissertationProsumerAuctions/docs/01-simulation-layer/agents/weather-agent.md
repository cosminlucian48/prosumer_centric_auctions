# WeatherAgent

## Overview

The `WeatherAgent` simulates environmental conditions that affect energy generation. It publishes weather updates that influence solar and wind generation rates for prosumers.

## Responsibilities

- Simulates environmental conditions
- Publishes weather updates (affects solar/wind generation)
- Influences generation rates for prosumers
- Provides weather data for simulation scenarios
