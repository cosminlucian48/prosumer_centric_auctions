# Data Models

This directory contains documentation for all data models used in the simulation layer.

## Models

- **[EnergyMarketPriceDataModel](./energy-market-price-data-model.md)** - Grid energy price data
- **[EnergyTransaction](./energy-transaction.md)** - Energy transaction records
- **[ProsumerEnergyBuyer](./prosumer-energy-buyer.md)** - Buyer information for auctions
- **[ProsumerEnergySeller](./prosumer-energy-seller.md)** - Seller information for auctions
- **[ProsumerGeneratorDataModel](./prosumer-generator-data-model.md)** - Generation data from database
- **[ProsumerLoadDataModel](./prosumer-load-data-model.md)** - Load data from database
